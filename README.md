## 📌 What is StreamXBot?

StreamXBot is a self-hosted bot that lets you listen to music inside your web browser. It connects with Telegram but works even if you don’t have a Telegram account. You can stream your favorite songs with no extra apps or software. Everything happens online, using your PC as a server.

Topics covered: music player, streaming audio, telegram, telegram bot.

## 💻 System Requirements

To run StreamXBot smoothly on your Windows computer, make sure you have:

## 🚀 Setting Up StreamXBot for the First Time

When you open StreamXBot, the app will guide you through initial setup.

1. **Configure your streaming port:** Usually, the default port (e.g., 8080) works fine. If you know what ports are, you can change this. Otherwise, leave it as is.

2. **Connect your Telegram bot token:** If you want to link StreamXBot with Telegram services, you need a Telegram bot token. If you don’t have one, the app will let you skip this step.

3. **Select music source:** The app streams music via browser. Select the options available within StreamXBot for playlists or files.

---

## 🎶 Using StreamXBot

Once the app runs, use your browser to open the streaming page. The URL will look like this:

```
http://localhost:8080
```

Replace `8080` if you used a different port during setup.

On this page, you can:

- Search for songs
- Play or pause music
- Create and save playlists
- Control volume and playback speed

You won’t need Telegram to listen, but if you connect your Telegram bot, you can control playback through Telegram messages or commands.

## 🛠️ Troubleshooting Common Issues

- **App does not start:** Make sure Windows is up to date. Try running the app as Administrator (right-click > Run as Administrator).
- **Browser won’t open streaming page:** Confirm you entered the right address (`http://localhost:8080`). Check if your firewall is blocking the app.
- **Music does not play:** Verify your internet connection and music sources are set up correctly.
- **Telegram bot features not working:** Double-check your bot token and connection settings.
